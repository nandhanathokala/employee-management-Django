# Employee Management System (Django)

##  Project Overview
A web-based Employee Management System developed using Django.  
The application provides secure authentication, a user dashboard, and complete CRUD functionality for managing departments and employees.



##  Features
- User Authentication (Login, Logout, Change Password)
- Secure Dashboard
- Department Management (Add, View)
- Employee Management (Add, View, Edit, Delete)
- Relational Database Design using Django ORM
- Admin Panel for data management



##  Technologies Used
- Python 3.7
- Django 2.1
- SQLite Database
- HTML (Basic UI)



##  Project Modules
- Accounts – Authentication and profile management
- Departments – Department creation and listing
- Employees – Employee CRUD operations
- Templates – HTML templates for UI
