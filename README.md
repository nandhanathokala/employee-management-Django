\# Employee Management System (Django)



\## Description

A Django-based web application with authentication, dashboard, and full CRUD operations for Departments and Employees.



\## Features

\- User authentication (Login, Logout, Change Password)

\- Dashboard with More Menu

\- Department Management (Add, View)

\- Employee Management (Add, View, Edit, Delete)



\## Technologies Used

\- Python 3.7

\- Django 2.1

\- SQLite

\- HTML
